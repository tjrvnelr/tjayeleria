# Tjay ELeria — Personal Portfolio

A modern, responsive personal portfolio built with Vite, React, TypeScript and Tailwind CSS. Showcases projects, experience, skills, resume and contact information.

## Features

- Hero with animated roles and social links
- About, Skills, Projects, Experience, Resume, and Contact sections
- Smooth scrolling and animated UI (Framer Motion)
- Downloadable resume (resume.pdf)

## Tech Stack

- React, TypeScript, Vite
- Tailwind CSS, Framer Motion
- Supabase (client), React Icons, Lucide
- Optional: Vercel for deployment

## Getting Started

Prerequisites
- Node.js (18+)
- npm or pnpm

Install

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build for production

```bash
npm run build
npm run preview
```

Linting & type checking

```bash
npm run lint
npm run typecheck
```

## Project Structure

- src/: React app source
- src/components/: site sections (Hero, About, Skills, Projects, etc.)
- public/: static assets (resume.pdf, images)

## Deployment

This site is ready for static deployment. Recommended hosts: Vercel, Netlify, or any static host.
